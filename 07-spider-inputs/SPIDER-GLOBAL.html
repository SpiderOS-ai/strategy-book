Spider -- Le Document Final

Pas un pitch deck. Pas un executive summary. C'est le document qu'on donne a un Thiel, un Graham, ou un co-fondateur pour qu'il comprenne TOUT. Derniere mise a jour : 19 fevrier 2026.

I. La These

L'Europe est en train de mourir

3 guerres. 3 defaites. Tech : zero GAFAM, zero plateforme. Industrie : les savoir-faire sont partis en Chine. Capital : les pepites se vendent aux Americains avant la taille critique.

Il reste un actif. Un seul. 25 millions de PME/ETI. 5 a 7 trillions d'euros de valeur ajoutee. 40% du PIB europeen. Et ces PME possedent quelque chose que personne n'a reussi a leur prendre : le tacite. Le relationnel tisse sur des decennies. Le savoir-faire depose dans les gestes par 30 ans de metier. Le sur-mesure accumule dans des milliers d'exceptions.

C'est le dernier tresor europeen. Ce que les Americains ne peuvent pas coder. Ce que les Chinois ne peuvent pas fabriquer. Et ce tresor pourrit de l'interieur.

Les 10 theses -- 4 axes

Axe 1 -- L'ACTIF

T1 : Le tacite est le dernier tresor. Le technicien qui "sent" la machine. Le commercial qui sait qu'on ne parle jamais du concurrent X devant ce client. 3 formes : relationnel (tissu de confiance), savoir-faire (intelligence des mains), sur-mesure (l'anti-process). C'est le seul actif non-reproductible. Et il meurt a chaque depart.

T2 : La vente B2B est un mensonge. Le dirigeant sait a qui il veut acheter avant que le process commence. L'appel d'offres = theatre. Le mot que personne ne prononce : confiance. Les CRM capturent le theatre. Le commercial garde le vrai dans sa tete.

T3 : Les US nous ont impose leurs regles. Salesforce, HubSpot, Monday, Notion -- 100% de la stack B2B concue pour les Americains. Mentalite US : transactionnelle. Mentalite europeenne : relationnelle. Spider est le premier outil concu POUR l'Europe.

Axe 2 -- L'IMPASSE

T4 : Ils veulent la couronne, pas le fardeau. Le familial a deja gagne. Le corporate veut son "dernier chapitre". L'heritier a recu le titre, pas le savoir. 3 profils, une maladie : ils veulent regner, pas gouverner.

T5 : Le ratio est catastrophique. CRM = effort max, benefice nul. ERP = 18 mois de migration = humiliation publique. Consultants = rapport dans le tiroir. Spider inverse le ratio : contrainte quasi-nulle, benefice reel, aveu zero.

Axe 3 -- LA BRECHE

T6 : L'IA rend possible ce qui etait un reve. Avant 2023, capturer le tacite = techniquement impossible. L'Europeen qui comprend le probleme n'a pas la tech. L'Americain qui a la tech ne comprend pas le probleme. Spider a les deux.

T7 : Triple convergence. 24 mois. Crise (les clients ne renouvellent plus) + Transmission (700 000 departs) + IA mature. Les trois ensemble = breche. Avant 2026 = pas assez mal. Apres 2028 = commoditise.

T8 : Le code est mort. Le service est le produit. Software-as-a-Service -> Service-as-Software. Le logiciel FAIT le travail. Palantir $400Mds vs Accenture $150Mds. Le seul monopole durable : la data captive.

Axe 4 -- Le Monopole

T9 : Le Shogunat Digital

Au Japon medieval, l'Empereur regnait (ceremonies, palais, respect) et le Shogun gouvernait (armee, impots, decisions). Chacun avait ce qu'il voulait. Pendant 700 ans.

Spider est le Shogun. Le dirigeant garde le titre, le bureau, la signature finale. Spider prend le pouvoir reel : capture, analyse, prepare, suggere. Le CEO valide. "Oui." "Oui." "Oui." Il croit decider. Il signe ce qu'on lui prepare. Ce n'est pas de la manipulation -- c'est du soulagement.

II. Le Modele

Service-as-Software

On ne vend pas un logiciel. On ne vend pas du conseil. On vend un resultat. Le ratio humain/IA evolue : 80/20 aujourd'hui -> 50/50 en phase intermediaire -> 20/80 en phase cible.

Architecture 2 couches

Couche 1 -- L'Assistant (reactif) : Chatbot IA connecte aux donnees business. Le dirigeant pose des questions. L'assistant diagnostique, revele des problemes, recommande des agents. C'est le "diagnostic gratuit" qui cree la douleur consciente.

  • La fenetre de 24 mois -- pour qui ? France d'abord (3-5 ans) ou monde (en retard) ?

  • SaaS vs ESN ? Si l'humain reste a 20%, valorisation ESN (1-3x). Si 5%, SaaS (10-20x).
  • Le ratio FDE:client. S'ameliore-t-il ? Toute la these repose dessus.
